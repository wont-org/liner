# lint

rules for project

## commit lint

> commit 规范: 采用 angular 风格, 借助 commitizen 进行交互式提交、commitlint 进行消息格式的校验。

```bash
# global
npm i commitizen -g
git cz
# npm
npm i commitizen -D
npx git-cz
```
